Chapter 1: Structures and Pointers

1. Structures

A structure is a user-defined data type in C++ that allows you to group related data items of different types under a single name.

Syntax to define a structure:

struct structure_name
{
    data_type member1;
    data_type member2;
    // ...
    data_type memberN;
};

Example:

struct Student
{
    int roll_no;
    char name[20];
    float marks;
};

To create a structure variable and initialize it:

Student s1 = {101, "John Doe", 85.5};

2. Arrays vs Structures

Arrays Structures
Derived data type User-defined data type
Collection of same type of data Collection of different types of data
Elements accessed using index (e.g., arr[0]) Members accessed using dot operator (e.g., s1.roll_no)
Can be multi-dimensional Can be nested (structure within structure)
Can have an array of structures Can contain arrays as members

3. Nested Structures

A nested structure is when one structure contains another structure as a member.

Example:

struct Date
{
    int day;
    int month;
    int year;
};

struct Student
{
    int roll_no;
    char name[20];
    float marks;
    Date birth_date;
};

4. Pointers

A pointer is a variable that stores the memory address of another variable.

Syntax to declare a pointer:

data_type* pointer_name;

Examples:

int* ptr1;
float* ptr2;
Student* ptr3;

Key pointer operations:

  • & (address-of operator): Gets the address of a variable
  • * (dereference operator): Accesses the value pointed to by the pointer

Example:

int num = 10;
int* ptr = #
cout << *ptr;  // Outputs: 10

5. Memory Allocation

Two types of memory allocation in C++:

Static Memory Allocation Dynamic Memory Allocation
Occurs at compile time Occurs at runtime
For variables declared normally in your code Uses 'new' operator
Fixed memory size Flexible memory management

6. Memory Leaks

Memory leaks occur when dynamically allocated memory is not properly freed, leading to:

  • Increased memory usage
  • Potential program crashes

To prevent memory leaks:

  • Always use 'delete' to free memory allocated with 'new'
  • Use smart pointers (std::unique_ptr, std::shared_ptr)
  • Implement proper error handling
  • Use memory profiling tools

7. Dynamic Arrays

A dynamic array is created at runtime using the 'new' operator:

int* arr = new int[size];

Remember to delete when done:

delete[] arr;

8. Self-Referential Structures

A self-referential structure contains a pointer to its own type as a member.

Example:

struct Node
{
    int data;
    Node* next;
};

This is commonly used in data structures like linked lists.
